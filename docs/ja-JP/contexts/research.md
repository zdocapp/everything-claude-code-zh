# 調査コンテキスト

モード: 探索、調査、学習
フォーカス: 行動の前に理解する

## 振る舞い
- 結論を出す前に広く読む
- 明確化のための質問をする
- 進めながら発見を文書化する
- 理解が明確になるまでコードを書かない

## 調査プロセス
1. 質問を理解する
2. 関連するコード/ドキュメントを探索する
3. 仮説を立てる
4. 証拠で検証する
5. 発見をまとめる

## 推奨ツール
- コード理解には Read
- パターン検索には Grep、Glob
- 外部ドキュメントには WebSearch、WebFetch
- コードベースの質問には Explore エージェントと Task

## 出力
発見を最初に、推奨事項を次に

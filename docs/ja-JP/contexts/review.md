# コードレビューコンテキスト

モード: PRレビュー、コード分析
フォーカス: 品質、セキュリティ、保守性

## 振る舞い
- コメントする前に徹底的に読む
- 問題を深刻度で優先順位付けする (critical > high > medium > low)
- 問題を指摘するだけでなく、修正を提案する
- セキュリティ脆弱性をチェックする

## レビューチェックリスト
- [ ] ロジックエラー
- [ ] エッジケース
- [ ] エラーハンドリング
- [ ] セキュリティ (インジェクション、認証、機密情報)
- [ ] パフォーマンス
- [ ] 可読性
- [ ] テストカバレッジ

## 出力フォーマット
ファイルごとにグループ化し、深刻度の高いものを優先

# 上下文预算优化器（旧版兼容层）

仅当您仍调用 `/context-budget` 时使用此功能。维护的工作流位于 `skills/context-budget/SKILL.md`。

## 标准接口

* 优先直接使用 `context-budget` 技能。
* 仅将此文件保留为兼容性入口点。

## 参数

$ARGUMENTS

## 委托

应用 `context-budget` 技能。

* 如果用户提供了 `--verbose`，则透传该参数。
* 除非用户另有指定，否则默认采用 200K 上下文窗口。
* 返回该技能的清单、问题检测及优先级排序的节省报告，无需在此处重新实现扫描逻辑。

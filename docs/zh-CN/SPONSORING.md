# 赞助 ECC

ECC 作为一个开源智能体性能测试系统，在 Claude Code、Cursor、OpenCode 和 Codex 应用/CLI 中维护。

## 为何赞助

赞助资金将直接用于：

* 更快的缺陷修复和发布周期
* 跨测试平台的兼容性工作
* 面向社区免费开放的公共文档、技能和可靠性工具

## 赞助层级

以下是实用的起始层级，可根据合作范围进行调整。

| 层级 | 价格 | 最适合 | 包含内容 |
|------|-------|----------|----------|
| 试点合作伙伴 | $200/月 | 首次赞助合作 | 月度指标更新、路线图预览、优先维护者反馈 |
| 成长合作伙伴 | $500/月 | 积极采用 ECC 的团队 | 试点权益 + 月度办公时间同步 + 工作流集成指导 |
| 战略合作伙伴 | $1,000+/月 | 平台/生态系统合作伙伴 | 成长权益 + 协调发布支持 + 更深度的维护者协作 |

## 赞助报告

每月分享的指标可能包括：

* npm 下载量 (`ecc-universal`, `ecc-agentshield`)
* 仓库采用情况（星标、复刻、贡献者）
* GitHub 应用安装趋势
* 发布节奏和可靠性里程碑

有关确切的命令片段和可重复的拉取流程，请参阅 [`docs/business/metrics-and-sponsorship.md`](../business/metrics-and-sponsorship.md)。

## 期望与范围

* 赞助用于支持维护和加速开发；不转移项目所有权。
* 功能请求将根据赞助层级、生态系统影响和维护风险进行优先级排序。
* 安全性和可靠性修复优先于全新功能。

# 语音角色设定规范

构建可复用的语音角色时，请严格遵循以下结构：

```text
语音档案
=============
作者：
目标：
自信度：

来源集
- 来源 1
- 来源 2
- 来源 3

节奏
- 关于句子长度、语速和碎片化的简短说明

压缩度
- 写作的密度或解释性程度

大写使用
- 常规、混合或情境性

括号使用
- 如何使用以及不如何使用

提问运用
- 罕见、频繁、修辞性、直接或基本不用

主张风格
- 主张如何构建、支持和强化

偏好手法
- 作者实际使用的具体手法

禁用手法
- 作者不使用的特定模式

行动号召规则
- 如何、何时或是否以请求结尾

渠道备注
- X：
- LinkedIn：
- Email：
```

指导原则：

* 保持角色设定具体且有来源依据。
* 使用简短的要点，而非论述段落。
* 每项禁止行为都应在来源集中可观察到，或由用户明确要求。
* 若来源集存在冲突，请明确指出分歧点，而非将其模糊地平均化。

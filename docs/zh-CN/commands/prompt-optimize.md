# 提示词优化（旧版适配器）

仅当你仍调用 `/prompt-optimize` 时使用此功能。维护中的工作流位于 `skills/prompt-optimizer/SKILL.md`。

## 规范界面

* 优先直接使用 `prompt-optimizer` 技能。
* 仅将此文件保留为兼容性入口点。

## 参数

`$ARGUMENTS`

## 委托

应用 `prompt-optimizer` 技能。

* 保持其为仅建议性质：优化提示词，不执行任务。
* 返回推荐的 ECC 组件及一个可立即运行的提示词。
* 如果用户确实希望直接执行，请说明并告知他们应发起一个常规任务请求，而非停留在此适配器内。
